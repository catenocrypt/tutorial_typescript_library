# tutorial_typescript_library
A simple TypeScrtipt library example

# Compile and publish

```shell
tsc
npm publish
```

# Usage

Add dependency

```shell
npm install --save cc-ts-lib-tutorial
```

use it like this:

```typescript
import { sayHello } from 'cc-ts-lib-tutorial';
sayHello();
```
